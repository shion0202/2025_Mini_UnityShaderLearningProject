# UnityShaderLearningProject

유니티 6 셰이더 학습 프로젝트 링크: [Github Link](https://github.com/shion0202/2026_Mini_UnityShaderLearningProject)

유니티 셰이더를 학습하고 이를 활용하여 프레임워크를 구현하는 개인 프로젝트입니다.<br>
당초 병행하던 팀 프로젝트와의 호환성을 고려해 Unity 2022 버전을 기준으로 학습을 진행했으나, 해당 프로젝트가 종료되고 Unity 6 기반의 새 프로젝트를 진행함에 따라 리포지토리 업데이트를 중단하고 새 리포지토리에서 이어서 학습할 예정입니다.
